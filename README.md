# BenzolPro
Goal: Create a high-converting, modern Dutch e-commerce website for selling various types of Benzol Lubricants engine oils (e.g., 5W30, 5W40, 10W40, 0W30). Target Audience: Dutch car enthusiasts and regular drivers looking for high-quality motor oil. Design & Vibe: > * Futuristic, highly appealing, and modern. Clean layout with high-quality visuals, but simple to navigate. Minimalistic yet striking color palette that fits a premium lubricant brand. The two main colours should be black and neon yellow. Key Features & UX: Ultra-fast checkout: Even the laziest visitors should be able to place an order within a few seconds (e.g., quick-buy buttons, minimal form fields, integration with Apple Pay/iDEAL). Smart filtering: A prominent filter or search tool based on oil viscosity (5W30, 5W40, etc.) so users instantly find the right product. License plate checking: customers should be able to enter their license plate in the homescreen and immediately they will get the recommended engine oil for their car. This should be integrated with the official Dutch authority named RDW. Language: The final website copy needs to be in Dutch, but users should be able to switch between dutch, english, polish, arabic and turkish easily. It should be able to work on different types of screens, like laptops but also small phonescreens.
